# mnd.app Project Description and Updates

## Project Description

mnd.app is an AI-powered study platform that helps students understand their notes faster.

Core goals:
- Upload notes
- Get simple explanations
- Generate summaries, quizzes, and flashcards
- Ask questions based on uploaded notes

Tech stack:
- Next.js (App Router)
- React
- Tailwind CSS
- OpenAI API

## Current Status

- Landing page and dashboard are live.
- Upload notes + AI explanation flow is implemented.
- Auth foundation is implemented:
  - Email/password signup and login
  - Session auth
  - Profile and settings pages
  - Email verification link flow
  - Provider login buttons (Google, Apple, Phone, Facebook, Discord) with starter routes
